# Jeu d'Echec

Jeu d'échec en python fait en binôme en terminal NSI.


## Installation

### Installer python 3.11.8 (https://www.python.org/downloads/release/python-3118/)

### Installation du module pillow

    pip install pillow
### Installation du module Chess

    pip install chess
